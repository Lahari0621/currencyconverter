# 💱 Currency Converter Web App

## 📌 Overview
The Currency Converter Web App is a simple and interactive web application that allows users to convert currencies using real-time exchange rates. It provides fast, accurate, and user-friendly conversion for travel, online shopping, and financial calculations.

---

## 🎯 Key Features
- 💱 Real-time currency conversion  
- 🌍 Supports multiple currencies (USD, INR, EUR, etc.)  
- ⚡ Instant and accurate results  
- 🖥️ Clean and user-friendly interface  
- 🔄 Dynamic updates based on user input  

---

## 🛠️ Technologies Used
- HTML  
- CSS  
- JavaScript  
- Exchange Rate API  

---

## ⚙️ How It Works
1. User selects base and target currency  
2. Enters the amount  
3. App fetches live exchange rates using API  
4. Converted value is displayed instantly  

---

## ▶️ How to Run the Project

```bash
git clone https://github.com/Lahari0621/currencyconverter.git
cd currencyconverter
```

### ▶️ Run the App
- Open `index.html` in your browser  
OR  
- Use Live Server (recommended)

## 📊 Output
The application displays:
- Converted currency value
