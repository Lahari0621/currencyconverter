# 💱 Currency Converter Web App

## 📌 Overview
The Currency Converter Web App is a simple and interactive web application that allows users to convert currencies using real-time exchange rates. It is designed to provide fast, accurate, and user-friendly currency conversion for everyday use such as travel, online shopping, and financial calculations.

---

## 🎯 Key Features
- 💱 Real-time currency conversion
- 🌍 Supports multiple currencies (USD, INR, EUR, etc.)
- ⚡ Instant and accurate results
- 🖥️ Clean and user-friendly interface
- 🔄 Dynamic updates based on user input

---

## 🛠️ Technologies Used
- HTML  
- CSS  
- JavaScript  
- Exchange Rate API  

---

## ⚙️ How It Works
1. User selects base currency and target currency  
2. Enters the amount to convert  
3. Application fetches live exchange rates using API  
4. Converted value is displayed instantly
